How to Use the MLB DFS Lineup Builder

This tool is designed to help you analyze MLB player data and generate optimal Daily Fantasy Sports (DFS) lineups for DraftKings. It supports both Classic and Showdown slates. Follow these steps to use it effectively:

  1. Select a Slate: Start by choosing a slate from the "Slate" dropdown. Slates are loaded and the earliest available slate is selected by default. Options include Classic (full slate) or Showdown (single game) formats. Note: For Showdown slates, the lineup is CPT (1.5x points) + 5 UTIL; for Classic, it is 2P + C + 1B + 2B + 3B + SS + 3 OF + UTIL.
  2. Filter Players: Use the "Search Player," "Position," and "Team" dropdowns to filter the player pool. You can select multiple positions or teams. The table below will update to show only matching players.
  3. Adjust Global Settings:
    • Max Player Exposure (%): Set the maximum percentage of lineups any single player can appear in (default 100%).
    • Lineup Salary Range ($): Adjust the dual slider to set the minimum and maximum total salary for generated lineups ($50,000 cap for both Classic and Showdown).
    • Total Projected Ownership Range (%): Adjust the dual slider to set the minimum and maximum total projected ownership percentage for generated lineups.
    • Number of Lineups: Specify how many lineups to generate (up to 500).
    • Only include players with Projected FP > 0: Check this to exclude players with zero projected fantasy points.
  4. Customize Player Projections and Exposures: In the player table:
    • Double-click on a player's "Projected FP" to edit their projection. For Captain slots in Showdown, this adjusts the base projection (multiplied by 1.5 internally).
    • Set per-player "Min Exposure" and "Max Exposure" percentages in the table columns.
  5. Set Stacks and Groups:
    • Click "Configure Stacks" to open the stack settings modal.
    • For stacks, select a "Stack Captain/CPT" and a "Stack Teammate" (hitter from the same team), and set the percentage of builds to include this stack.
    • For groups, add rules by selecting "At Least" or "At Most", the number, select players, optional "When lineups include" player, and percentage of builds, then add to the list.
  6. Generate Lineups: Click "Run" to optimize and generate lineups using linear programming. The tool maximizes projected points while respecting salary, ownership, positions, exposures, stacks, and group constraints.
  7. View and Download Results: Generated lineups appear in a table below. For Showdown: CPT + 5 UTIL; for Classic: P, P, C, 1B, 2B, 3B, SS, OF, OF, OF, UTIL. Download as CSV for easy import into DraftKings. A second table shows the desired vs. actual exposure and stack inclusion.
  8. Tips:
    • The tool adds randomness (up to ±50%) to projections for diversity and increases it if constraints are hard to meet.
    • If min exposures, stacks, or group constraints aren't met, an alert will show with details.
    • Toggle Dark Mode in the top-right for better visibility.
This tool helps you analyze MLB player data and generate optimal fantasy baseball lineups for DraftKings. Use the filters to select teams, positions, and more. Adjust settings like salary range ($50,000 cap), total projected ownership range, exposure limits, and number of lineups, then click 'Run' to create lineups based on projected fantasy points, salary, and ownership constraints. Click table headers to sort data. For Showdown slates, lineups include 1 Captain (1.5x points) and 5 UTIL players; for Classic slates: 2 Pitchers + C + 1B + 2B + 3B + SS + 3 OF + UTIL. You can define stacks (Captain/CPT + Teammate from same team) and groups for a specified percentage of builds. Data is from your CHEAT SHEET sheet.
